HOW TO SET UP DIRECTORY TO RUN THIS CODE

	1. Mkdir myapp
	2. Cd myapp
	3. Npm init
	4. Npm install express joi mongodb mongoose
	5. Npm install -g nodemon

Then copy/paste the folder "Schemas" and all the .js files into the main directory.

"nodemon express.js" to run an example of express
"nodemon export\ test.js" to run a simple script which will save
